Introduccion a Bootstrap

Categoria: Programacion

En este curso aprenderemos sobre bootstrap un popular framework de diseño front-end que facilita la creación de sitios web y aplicaciones web responsivas y de alta calidad. Desarrollado por Twitter, Bootstrap proporciona una serie de herramientas y componentes predefinidos que permiten construir interfaces de usuario modernas y funcionales con rapidez y eficiencia.
